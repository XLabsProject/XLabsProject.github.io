IW6x CLIENT SUPPORT

Below are guides to help you with any client issues you may have with IW6x.

Experiencing errors? View our error page HERE.

IW6x CLIENT INSTALL GUIDE

If you do not own the game on Steam

Requirements:

  • A Torrent Client. (We recommend qBittorrent)
  • The IW6x Full Game torrent
  • The IW6x Client

Getting Started:

  1. Download the torrent which contains the full game including the DLC content.
  2. Move the folder to a location like C:\Games\IW6x.
  3. Place the IW6x.exe file you downloaded from the link above and place it in the IW6x folder.
  4. To start the game, double-click iw6x.exe and a menu will appear asking you to choose between The Singleplayer or Multiplayer. Choose Multiplayer and the game should lauch.

To change your name, open the in-game console (press the ~ key) and type /name yourname.

To unlock all, open the in-game console (press the ~ key) and type /unlockall.

Custom classes are made in the Private Match section.

IW6x CLIENT/DLC INSTALL GUIDE

If you do own the game on Steam, and want DLC.

Requirements:

  • IMPORTANT: This only works for English copies of the game. Make sure you have your game in english before preceding.
  • A torrent client. (We recommend qBittorrent)
  • The IW6x DLCs torrent
  • The IW6x Client

Getting Started:

  1. Download the torrent which contains the DLC content.
  2. Move the completed download into your Steam Ghosts Folder.
  3. Place the iw6x.exe in the game folder.
  4. To start the game, double-click iw6x.exe and a menu will appear asking you to choose between The Singleplayer or Multiplayer. Choose Multiplayer and the game should lauch.

IW6x Controller Guide

Text Guide:

IW6x should support xbox controllers out of the box. Make sure gamepad support is enabled in settings.

PS4

  1. Download DS4Windows, make sure you pick DS4Windows_xxx_x64.zip
  2. Install/Extract it to a safe place and open it
  3. It should detect your controller, but if it does not you may need to install the driver.
  4. Check if gamepad works in IW6x.

IW6x Private Match Guide

Method 1 - (Friends List):

  1. Note: Requires you to have each other added as friends on Steam and to have Steam running.
  2. Start IW6x and select 'CREATE GAME' from the main menu and click 'START GAME'
  3. Tell your friends to press the F key on the main menu to bring up the in-game friends list
  4. Tell them to double click your name to join your game (If they hear a chicken noise, see "Failed to join")

Method 2 - (Connect Command):

  1. Go to here and PM anyone who wants to join your IP Address
  2. Start IW6x and launch a private match.
  3. Tell anyone who wants to join, open the console by using the tilde key while at the main menu
  4. Get them to enter the following command, replacing 'HostsIPAddress' with the IP you previously sent him:
  5. /connect HostsIPAddress:27016

Failed to join

  1. If you followed the above correctly and your friends are unable to join them you the host of the private match will need to port forward in order for them to connect to you.
  2. Port Forward port 27016 UDP & TCP
  3. You can find how to do this by Googling 'YourRouterName Port Forwarding'
  4. Note: If port forwarding is too complex, just stick to playing public servers, or maybe join an empty server so you have the same experience as a private game.
