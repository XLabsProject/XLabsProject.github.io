Console Commands

Frequently asked questions related to console commands for our clients.

Commands Example Cheat Protected IW4x IW6x Description
unlockstats unlockstats X ✓ X Unlock everything (IW4x) and rank up.
unlockall unlockall X X ✓ Unlock everything (IW6x) and rank up.
cg_fov cg_fov 90 (Changes FOV to 90) X ✓ X Manually tweak your desired FOV value.
sensitivity sensitivity 1.25 X ✓ ✓ The command is useful for fine tuning of your desired sensitivity.
cg_drawfps cg_drawfps 1 (enables) cg_drawfps 0 (disables) X ✓ ✓ Displays a FPS counter on-screen.
cl_yawspeed cl_yawspeed 800 X ✓ ✓ Adjusts the yawspeed (Turn Right/Left) sensitivity.
customtitle customtitle ^1Hello! X ✓ X Adjusts your title with whatever you want.
com_maxfps com_maxfps "85" X ✓ ✓ Adjusts the FPS (this is only useful for doing a specific FPS)
connect connect 127.0.0.1:27017 (IP:Port) X ✓ ✓ Connect to specified IP and port.
map mp_mapname map mp_afghan X ✓ ✓ Change the map.
devmap mp_mapname devmap mp_rust - ✓ X Change the map and enables cheats.
sv_cheats sv_cheats 1(enables), sv_cheats 0 (disables) - ✓ ✓ Enable cheats.
fast_restart fast_restart X ✓ ✓ Quickly restarts the match.
map_restart map_restart X ✓ X Fully restart the map.
status status X ✓ ✓ Display information about the current game and players.
clientkick clientkick 1 X ✓ ✓ Kicks the player by ID. Use the status command to obtain the client ID.
g_speed g_speed 300 ✓ ✓ ✓ Change game speed (requires devmap or sv_cheats 1) [Default = 190]
g_gravity g_gravity 300 (gravity is reduced by half) ✓ ✓ ✓ Change game gravity (requires devmap or sv_cheats 1) [Default = 600]
bg_fallDamageMaxHeight / bg_fallDamageMinHeight bg_fallDamageMaxHeight 300 ✓ ✓ X Changes amount you need to fall to take damage.
