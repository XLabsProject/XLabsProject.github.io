IW4x CLIENT INSTALL GUIDE

If you do not own the game on Steam

Requirements:

  • A Torrent Client. (We recommend qBittorrent)
  • The IW4x Full Game torrent
  • DirectX 9.0

Getting Started:

  1. Download the torrent which contains the full game including the DLC content.
  2. Move the folder to a location like C:\Games\IW4x.
  3. To start the game, double-click IW4x.exe.

To change your name, open the in-game console (press the ~ key) and type /name yourname.

To unlock all, open the in-game console (press the ~ key) and type /unlockstats.

IW4x STEAM INSTALL GUIDE

If you do own the game on Steam, and want DLC/IW4x.

Requirements:

  • A Torrent Client. (We recommend qBittorrent)
  • The IW4x Client Files
  • The IW4x DLC Files

Getting Started:

  1. Download the torrent which contains the DLC content.
  2. Move the completed download into your Steam MW2 Folder.
  3. Place the extracted client files in the game folder.
  4. To start the game, double-click IW4x.exe.
